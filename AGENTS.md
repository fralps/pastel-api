# Project AI Agents

This repository provides a specialized agent for the Ruby on Rails API stack.

## Available Agent

- `rails-stack-guardian`: Ruby, Rails, RSpec, and RuboCop expert focused on quality, security, conventions, and testability.

## Location

The agent file is located at `.github/agents/rails-stack-guardian.agent.md`.

## When to Use It

- Add or modify Rails endpoints.
- Review a service object or business logic.
- Write or fix RSpec specs (models, requests, serializers, mailers).
- Clean up code to satisfy RuboCop without changing behavior.

## What It Prioritizes

- Business correctness and API compatibility.
- Security (auth, input validation, explicit errors).
- Idiomatic Ruby readability.
- Robust and relevant test coverage.
- Compliance with Rails and RuboCop conventions.
